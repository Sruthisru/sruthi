To install the git
******************
sudo apt get install git-core
sudo apt-get install git-doc
sudo apt-get install git-svn
sudo apt-get build-dep git-core git-doc git-svn

if the above commands are failed, use
*************************************
sudo apt-get update
sudo apt-get build-dep git-core git-doc git-svn
